# Tailwind Tokenless

Tailwind CSS without tokens.

## Getting started

### Installation

```zsh
npm install @nattui/tailwind-tokenless
```

### Usage

```css
@import "tailwindcss";
@import "@nattui/tailwind-tokenless";

html {
  font-size: 10px;
}
```

### Root font size 10px

`apps/styles/global.css`

```css
html {
  font-size: 10px;
}
```

`.vscode/settings.json`

```json
{
  "tailwindCSS.rootFontSize": 10
}
```
